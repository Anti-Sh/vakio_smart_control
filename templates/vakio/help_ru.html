Общие сведения

Модуль предназначен для взаимодействия с устройствами Vakio в режиме подключения к локальному MQTT-брокеру.

Возможности модуля:

  • управление устройствами из модуля;
  • управление устройствами через привязвнные свойства/методы;
  • управление устройствами через API.

Для взаимодействия с роутером используется MQTT.

В модуле имеется цикл, который принимает и публикует сведения по протоколу MQTT.

Для добавления интернет-центра необходимо войти на прибор по его IP адресу и в Настройках mqtt переключить на Свой сервер и прописать данные для подключению к вашему брокеру. Также, надо прописать настройки подключения в Настройках самого модуля. Далее, нажмите кнопку +Добавить, придумайте название прибора, выберите тип устройства и введите топик. Топик должен быть уникальный.Например, vakio/zal. После нажатия кнопки Добавить в списке появится устройство и его возможные свойства.

Обращение к устройству по API с помощью кода

Параметры:

  • name — имя прибора
  • topic — управляющий топик устройства из таблицы в предыдущем разделе
  • data — устанавливаемое значение

Примеры:

Отключение устройства: 
callAPI('/api/module/vakio','GET',array('name'=>[#NAME#],'topic'=>'state','data'=>'off'));
Режим приток MAX 
callAPI('/api/module/vakio','GET',array('name'=>[#NAME#],'topic'=>'workmode','data'=>'inflow_max'));
Скорость вентилятора 4 
callAPI('/api/module/vakio','GET',array('name'=>[#NAME#],'topic'=>'speed','data'=>'4'));
